# F1.1: Onboarding 5 Passos

**Skill**: Frontend Core + UI/UX  
**Estimativa**: 8h  
**Prioridade**: P0  
**Dependências**: nenhuma

## Descrição
Criar fluxo de onboarding de 5 passos para usuários novos chegarem do zero ao primeiro loop real em até 10 minutos.

## Subtasks
- [ ] F1.1.1 Criar schema de configuração inicial do usuário (concursos, data prova, tempo diário)
- [ ] F1.1.2 Implementar Step 1: Seleção de concurso alvo
- [ ] F1.1.3 Implementar Step 2: Data da prova (calcular fase automaticamente)
- [ ] F1.1.4 Implementar Step 3: Tempo disponível diário
- [ ] F1.1.5 Implementar Step 4: Primeira missão gerada
- [ ] F1.1.6 Implementar Step 5: Registro da primeira sessão
- [ ] F1.1.7 Criar lógica de retomar onboarding incompleto
- [ ] F1.1.8 Validar fluxo completo (zero → loop real em ≤10min)

## Critérios de Aceite
- [ ] Usuário consegue completar onboarding em ≤10 min
- [ ] Nenhum passo exige criação manual de card
- [ ] Onboarding pode ser retomado se incompleto
- [ ] Ao concluir, usuário já entra no loop real do produto
