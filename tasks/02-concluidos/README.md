# Tasks Concluídas

Esta pasta armazena as tasks que foram finalizadas.

## Como mover uma task para concluída

1. Marque todas as subtasks como completadas no arquivo da task
2. Mova o arquivo para esta pasta
3. Atualize o `README.md` principal se necessário

## Estrutura

```
02-concluidos/
└── [task-id]-[nome-da-task].md
```
