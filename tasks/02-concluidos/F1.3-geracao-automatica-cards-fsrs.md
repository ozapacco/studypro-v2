# F1.3: Geração Automática de Cards FSRS

**Skill**: FSRS Engine + Backend  
**Estimativa**: 4h  
**Prioridade**: P0  
**Dependências**: F1.2

## Descrição
Gerar cards FSRS automaticamente a partir de erros registrados nas sessões.

## Subtasks
- [ ] F1.3.1 Implementar biblioteca FSRS (scheduler, card state, intervals)
- [ ] F1.3.2 Criar Edge Function para gerar card a partir de erro
- [ ] F1.3.3 Implementar lógica de unicidade (tópico+matéria)
- [ ] F1.3.4 Adicionar contexto do erro no card ("Você errou este tópico em X das últimas Y sessões")
- [ ] F1.3.5 Configurar prioridade maior para cards automáticos
- [ ] F1.3.6 Testar geração em < 2 segundos

## Critérios de Aceite
- [ ] Card gerado em até 2s
- [ ] Duplicado não explode a base
- [ ] Contexto do erro aparece no card
- [ ] Sessão sem erro tagueado não gera card
