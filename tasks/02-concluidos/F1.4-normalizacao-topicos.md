# F1.4: Normalização de Tópicos

**Skill**: Topic Engine + Backend  
**Estimativa**: 5h  
**Prioridade**: P0  
**Dependências**: F1.2

## Descrição
Normalizar tags de erro para tópicos canônicos e rastrear desempenho por tópico.

## Subtasks
- [ ] F1.4.1 Criar tabela `topicPerformance`
- [ ] F1.4.2 Implementar normalizador de tópicos (mapping livre → canônico)
- [ ] F1.4.3 Criar dicionários de tópicos por matéria (Penal, Proc. Penal, Const., Adm., etc.)
- [ ] F1.4.4 Implementar autocomplete com sugestões de tópicos existentes
- [ ] F1.4.5 Criar sistema de aliases (fundir tags semelhantes)
- [ ] F1.4.6 Implementar fallback para criar novo tópico canônico

## Critérios de Aceite
- [ ] Usuário pode digitar livremente
- [ ] Sistema sugere tópico já existente antes de criar novo
- [ ] Admin/local engine pode fundir aliases depois
- [ ] Estatísticas usam sempre o tópico canônico
